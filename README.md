
# Sticky Notes

    Sticky Notes is Android based application which is used to save the Notes.
    The application will read the for you using the Text-To-Speech Engine.
    
### Database
* Room (SqLite)
* LiveData
### View 
* RecyclerView
### Read Note
* Text-To-Speech
